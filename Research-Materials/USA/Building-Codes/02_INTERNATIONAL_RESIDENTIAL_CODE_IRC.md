# INTERNATIONAL RESIDENTIAL CODE (IRC)

## Overview
The International Residential Code (IRC) is a comprehensive, stand-alone residential code that covers one- and two-family dwellings and townhouses up to three stories. It is the most widely used residential building code in the United States.

## Official Source
- **Publisher**: International Code Council (ICC)
- **Website**: https://www.iccsafe.org
- **Online Access**: https://codes.iccsafe.org (free viewing, registration required)
- **Current Edition**: 2024 IRC (updated every 3 years)
- **Previous Editions**: 2021, 2018, 2015, 2012, 2009, 2006, 2003, 2000

## Scope and Coverage
The IRC covers:
- One- and two-family dwellings
- Townhouses up to three stories above grade
- Live/work units
- Accessory structures

Buildings outside this scope fall under the International Building Code (IBC).

## Span Tables
The IRC includes extensive span tables for:
- Floor joists
- Ceiling joists
- Rafters
- Headers and beams
- Studs
- Girders

These tables eliminate the need for engineering calculations for standard residential construction.

## Key Advantages
1. **Self-contained**: One code covers all aspects
2. **Prescriptive**: Easy to use without engineering
3. **Span tables**: Pre-calculated for common applications
4. **Widely adopted**: Used throughout U.S.
5. **Regularly updated**: Current with technology
6. **User-friendly**: Organized for contractors and builders

## Key Differences from B&R Code
1. **Scope**: Residential only vs. all buildings
2. **Approach**: Prescriptive vs. primarily performance
3. **Integration**: All trades in one code
4. **Updates**: 3-year cycle vs. decades
5. **Accessibility**: Modern requirements included
6. **Energy**: Mandatory efficiency provisions
7. **Detail**: Extensive span tables and details
