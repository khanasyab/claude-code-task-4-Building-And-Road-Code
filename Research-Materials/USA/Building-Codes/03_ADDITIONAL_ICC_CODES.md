# ADDITIONAL INTERNATIONAL CODE COUNCIL (ICC) CODES

## Overview
In addition to the International Building Code (IBC) and International Residential Code (IRC), the International Code Council publishes a complete family of complementary codes known as the I-Codes. These codes work together to provide comprehensive coverage of building construction, fire safety, energy efficiency, and specialized systems.

## Official Source
- **Publisher**: International Code Council (ICC)
- **Website**: https://www.iccsafe.org
- **Online Access**: https://codes.iccsafe.org (free viewing, registration required)
- **Update Cycle**: 3 years (synchronized with IBC)

## How the I-Codes Work Together

### Hierarchy
1. **IBC or IRC**: Base building code
2. **Technical codes**: IMC, IPC, IFGC, IECC provide detailed requirements
3. **IFC**: Fire safety operations
4. **Specialty codes**: IEBC, IPMC, ISPSC, IWUIC, IgCC for specific situations

### Coordination
- Cross-referenced throughout
- Consistent terminology
- No conflicts between codes
- Synchronized update cycle
