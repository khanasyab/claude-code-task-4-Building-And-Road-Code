# INTERNATIONAL BUILDING CODE (IBC)

## Overview
The International Building Code (IBC) is a model building code developed by the International Code Council (ICC). It is the most widely adopted building code in the United States and provides minimum requirements for buildings using prescriptive and performance-related provisions.

## Official Source
- **Publisher**: International Code Council (ICC)
- **Website**: https://www.iccsafe.org
- **Online Access**: https://codes.iccsafe.org (free viewing, registration required)
- **Current Edition**: 2024 IBC (updated every 3 years)
- **Previous Editions**: 2021, 2018, 2015, 2012, 2009, 2006, 2003, 2000

## Scope and Coverage
The IBC addresses all buildings except detached one- and two-family dwellings and townhouses not more than three stories (covered by IRC).

## Key Differences from Historical B&R Codes
1. **Performance-based provisions**: IBC allows alternative compliance methods
2. **Seismic design**: Comprehensive seismic provisions throughout
3. **Accessibility**: Extensive requirements for disabled access
4. **Energy efficiency**: Integrated energy requirements
5. **Regular updates**: 3-year cycle vs. decades for older codes
6. **Consensus process**: Developed through public hearings and votes
