Chapter 20 Referenced Standards
User note:
About this chapter: Chapter 20 notes the standard that is referenced in this code. This chapter provides the location where future standards referenced by this code can be listed. The standard is part of the code to the extent of the reference to the standard. Compliance with the referenced standard is necessary for compliance with this code. By providing this specifically adopted standard, the construction and installation requirements necessary for compliance with the code can be readily determined. The basis for code compliance is, therefore, established and available on an equal basis to the building code official, contractor, designer and owner. This standard is listed herein by the promulgating agency of the standard, the standard identification, the effective date and title, and the sections of this document that reference the standard.
