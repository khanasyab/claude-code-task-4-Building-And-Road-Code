Chapter 14 Referenced Standards
User notes:
About this chapter:
This code contains numerous references to standards promulgated by other organizations that are used to provide requirements for materials and methods of construction. Chapter 14 contains a comprehensive list of all standards that are referenced in this code. These standards, in essence, are part of this code to the extent of the reference to the standard.
