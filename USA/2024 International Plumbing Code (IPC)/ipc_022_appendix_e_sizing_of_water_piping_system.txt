Appendix D Degree Day and Design Temperatures
This appendix is informative and is not part of the code.
User notes:
About this appendix: This code refers to the 97.5 percent winter design temperature for directing the code user to requirements for design of the plumbing system. Appendix D provides such temperatures for many major cities in the United States.
